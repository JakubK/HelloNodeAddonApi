## hello-world

Introduces methods:
- `hello` returning string "world"
- `add` returning sum of 2 given numbers
- `asArray` returning array of given numbers
## Getting started

### Prerequisites

1. Windows System
2. Visual Studio with C++ configuration

Without these, setup is likely to fail on configure step 

### Setup
1. npm install
2. npm run configure

### Development
1. npm run build
2. npm run test
